# Disciple Vertueux: 365 Objectifs de Dieu pour votre vie

## Description
E-book dévotionnel chrétien conçu pour un public général (hommes et femmes), avec 365 jours de réflexions bibliques, messages inspirants et espaces pour notes personnelles.

## Caractéristiques du Design

### Mise en page des Pages Quotidiennes
- **Design à 2 colonnes**: 
  - Colonne gauche: Texte biblique complet + commentaire/message
  - Colonne droite: Date + espace élargi pour notes personnelles

### Design des Dates
- **Carrés avec des couleurs pastel variées**:
  - Jour 1: Vert doux
  - Jour 2: Marron clair  
  - Jour 3: Orange pastel
  - Jour 4: Rose pâle
  - Jour 5: Bleu ciel
  - Jour 6: Lavande
  - Jour 7: Pêche
- Chaque carré contient:
  - Jour de l'année (1-365) mis en évidence en grand
  - Date réelle en dessous en taille plus petite

### Contenu Biblique
- **Textes bibliques complets** (pas seulement des références)
- Référence biblique après le texte
- Commentaires/messages inspirants dirigés vers le disciple chrétien
- Accent mis sur la perspective chrétienne des écritures

### Éléments Visuels
- Illustrations aquarelle minimalistes dessinées à la main
- Palette de couleurs pastel douces
- Police Georgia pour le texte principal
- Police Dancing Script pour les notes manuscrites
- Design entièrement réactif

## Structure du Prototype

### Contenu Inclus
1. **Page de couverture** avec illustration adaptée pour un public général (à modificar)
2. **Page d'introduction** avec message personnel pour le disciple chrétien
3. **Page d'introduction de janvier** avec espaces pour:
   - Objectifs du mois
   - Apprentissages attendus
   - Prière personnelle
   - **NOUVEAU: Suivi des Habitudes** avec calendrier interactif de 31 jours
   - **NOUVEAU: Auto-évaluation de la Croissance Spirituelle** avec barres de progression pour 4 catégories
4. **7 jours complets** (1-7 janvier) avec:
   - Lectures bibliques spécifiques avec texte complet
   - Messages personnels axés sur la perspective chrétienne
   - Espaces fonctionnels élargis pour notes
   - Notes manuscrites avec applications pratiques

### Nouvelles Fonctionnalités Interactives

#### Suivi des Habitudes
- **Titre**: "Mon Habitude du Mois"
- Espace éditable pour écrire l'habitude personnelle
- Calendrier avec 31 cases numérotées
- Fonctionnalité de clic pour marquer les jours complétés (✓)
- État sauvegardé automatiquement dans le navigateur

#### Auto-évaluation de la Croissance Spirituelle
- **Titre**: "Ma Croissance Spirituelle"
- 4 catégories d'évaluation:
  - **FOI** (barre de progression 1-10)
  - **DISPONIBILITÉ** (barre de progression 1-10)
  - **AMOUR DE SOI** (barre de progression 1-10)
  - **ÉQUILIBRE ÉMOTIONNEL** (barre de progression 1-10)
- Barres interactives qui se colorent au clic
- Dégradés pastel qui reflètent le niveau sélectionné
- État sauvegardé automatiquement dans le navigateur

## Comment Utiliser

### Version Numérique
1. Ouvrir `index.html` dans n'importe quel navigateur web moderne
2. Naviguer en faisant défiler ou en utilisant les commandes de navigation
3. Cliquer sur les espaces de notes pour écrire
4. Les notes se sauvegardent automatiquement dans le navigateur

### Version Imprimée
1. Ouvrir `index.html` dans le navigateur
2. Utiliser Ctrl+P (Cmd+P sur Mac) pour imprimer
3. Sélectionner les paramètres d'impression appropriés
4. Les espaces de notes sont prêts pour l'écriture manuelle
